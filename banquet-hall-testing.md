# 中式宴会厅页面测试指南

## 测试步骤

1. 在浏览器中打开 `banquet-hall.html` 页面
2. 使用 WASD 或方向键控制玩家角色移动
3. 尝试以下交互功能：
   - 点击"送红包"按钮，观察红包飞向中央花堂的动画效果
   - 点击"放烟花"按钮，观察烟花在顶部绽放的效果
   - 在聊天输入框中输入文字，按回车或点击"发送"按钮，观察玩家头像上的聊天气泡显示
   - 点击附近的宾客角色，发送祝福语并与他们互动

## 功能验证清单

### 场景渲染
- [ ] 等距视角的中式宴会厅场景正确显示
- [ ] 红色灯笼悬挂在场景四周
- [ ] 4-6张圆桌合理分布在场景中
- [ ] 红绸装饰元素正确显示
- [ ] 中央拜堂花堂作为焦点元素突出显示

### 角色系统
- [ ] 6-8个模拟宾客角色正确显示在场景中
- [ ] 玩家角色可以通过键盘控制移动
- [ ] 模拟宾客会随机显示聊天气泡（如"恭喜恭喜"、"百年好合"等）
- [ ] 玩家角色具有聊天气泡功能

### 互动功能
- [ ] 点击"送红包"按钮：
  - 红包动画从玩家位置飞向中央花堂
  - 所有宾客同时弹出"恭喜"气泡
- [ ] 点击"放烟花"按钮：
  - 顶部出现烟花粒子动画
  - 所有宾客头顶弹出"哇"表情
- [ ] 点击附近宾客：
  - 发送预设祝福语（如"新婚快乐"、"百年好合"）
  - 该宾客回复一个笑脸表情
- [ ] 聊天功能：
  - 输入框可以输入自定义文本
  - 按回车或点击发送按钮后，玩家头像显示输入内容

### 用户界面
- [ ] 控制按钮清晰可见且易于操作
- [ ] 聊天输入框和发送按钮布局合理
- [ ] 返回按钮功能正常
- [ ] 操作提示信息明确

### 性能和兼容性
- [ ] 页面加载速度快
- [ ] 动画流畅无卡顿
- [ ] 在不同屏幕尺寸下布局合理
- [ ] 浏览器兼容性良好

## 故障排除

如果遇到问题，请检查：
1. 所有资源文件路径是否正确
2. JavaScript是否有语法错误
3. CSS样式是否正确定义
4. WebSocket服务器是否正常运行（如果需要实时多人功能）
